Platform Mimarisi

Uctan uca gozlemlenebilir, guvenilir ve olceklendirilebilir

AirSense mimarisi; MQTT veri akisindan Redis Pub/Sub ve WebSocket yayina, oradan panel ve mobile kadar tek hat uzerinden calisir.

Backend Servisleri

FastAPI + MongoDB (motor) + JWT + MQTT worker + Redis onbellek/pubsub ile gercek zamanli veri isleme.

Dashboard Katmani

React 19 + TypeScript + Vite + Tailwind + Recharts ile canli panel, alarm gecmisi ve raporlama.

ML Motoru

Scikit-learn Isolation Forest ile anomali tespiti; NumPy/Pandas veri isleme ve model yonetimi.

Mobil ve Bildirim

Flutter + Provider + Dio + FCM ile push bildirim, canli veri ve saha odakli mudahale akislar.

Dijital Ikiz Motoru

FastAPI backend + React/Three.js 3D gorunum, gaz sizintisi ve tahliye simulasyonlari.

Veri akisi: ESP32 -> MQTT Broker -> Worker -> MongoDB + Redis -> WebSocket Gecidi -> Panel/Mobil.
